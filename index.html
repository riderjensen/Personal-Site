Inc Fast Now

Inc Fast Now is a client project that I was contracted to make. The frontend is built with React.js while the backend is a Node.js RESTful API. The code includes cart management, user validation, and payment processing. It also included the creation of a CRM, mail server, and CMS website. I handled the whole project from start to finish including server set up and management. Email or call me for access to review the source code.

Project Arcanite

Project Arcanite is a posting platform similar to Yik-Yak with a few differences. The frontend is built with React.js and the backend is a restful Node.js application. It includes handwritten authentication as well as JWT verification. Source
