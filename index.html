Projects

This is just a small showcase of the projects I have worked on. View my Github to see all my projects and source code.

Inc Fast Now

Inc Fast Now is a client project that I was contracted to make. The frontend is built with React.js while the backend is a Node.js RESTful API. The project included cart management, user validation, payment processing, a CRM, mail server, and CMS website.

URL Shortner

This URL shortner was built as a RESTful api with an optional MVC, made in Node.js. Its totally open for people to download and use themselves. It's really easy to add in your own middleware and is still being updated with new technologies. Source

National Asset Protection

National Asset Protection is a client site built on Wordpress. I worked in conjunction with a third party designer in order to create this performant site. It includes a number of custom components that were made just to fit the design.

Project Arcanite

Project Arcanite is a posting platform similar to Yik-Yak with a few differences. The frontend is built with React.js and the backend is a restful Node.js application. It includes handwritten authentication as well as JWT verification. Source

Advanced Tax Group

Advanced Tax Group is a client Wordpress site. I worked with a designer to create the original idea and then we pivoted the site to become revenue generating by adding more content and creating a subscription model in order to view that content.

Lean List

Lean List is built with Dart and Flutter, it's a native mobile application that allows for multiple people to create, keep, and edit lists with other people. Because this app's backend is located on Firebase, it is cross platform. Source
